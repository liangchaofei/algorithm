## 狄克斯特拉算法
+ 适用于有向无环图
+ 理念：找出图中最便宜的节点，并确保没有到该节点到更便宜的路径

### 加权图
+ 带权重
+ 计算加权图最短路径，使用狄克斯特拉算法

### 非加权图
+ 不带权重
+ 计算非加权图带最短路径，可使用广度优先搜索算法
