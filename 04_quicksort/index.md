## 快速排序
+ 时间复杂度：NlogN


## python
+ 数组切片
```python
    arr = [1,2,3]
    a = arr[1:] 
```

+ 三元符
```python
    条件为真的结果 if 条件 else 条件为假的结果
    arr[0] if arr[0] > arr[1] else arr[1]
```
+ 向数组中添加元素
```python
    arr.append(item)
```
